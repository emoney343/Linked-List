# Linked List Contact Manager
 
A command-line contact manager written in Java. Built as a hands-on data structures project — the list is implemented from scratch without using any built-in Java collections.
 
## How to Run
 
```bash
javac *.java
java Test
```
 
## Features
 
- Add contacts to the beginning, end, or a specific index
- Search, view, and remove contacts by name
- Edit any field on an existing contact
- Input validation with descriptive error messages
 
## What I Learned
 
- How a singly linked list works under the hood - nodes, head references, and pointer manipulation
- How to implement common list operations (insert, remove, traverse) at the code level rather than relying on built-in libraries
- How to separate responsibilities across classes: the list handles storage, the manager handles business logic, and the UI handles user input
- How indexed insertion and removal work, and why they are O(n) in a linked list
